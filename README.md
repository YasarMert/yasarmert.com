# My website: [yasarmert.com](https://yasarmert.com)
The source files for my website (to host with GitHub Pages)

## How it is setup
1) This website is made locally using <a href="https://jekyllrb.com/" target="_blank">Jekyll</a> and <a href="https://github.com/jglovier/resume-template" target="_blank">jglovier's repo</a>.
2) After the website is created, it is uploaded to GitHub.
3) When the repository is ready, GitHub Pages are activated (with the custom domain, [yasarmert.com](https://yasarmert.com).)
4) The website is now available at [yasarmert.com](https://yasarmert.com)!
